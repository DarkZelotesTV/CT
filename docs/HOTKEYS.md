# Hotkeys

Die wichtigsten Hotkeys des Clients werden zentral im `MainLayout` registriert und können im User-Settings-Dialog (Tab "Hotkeys") angepasst oder deaktiviert werden.

| Aktion | Standard | Hinweise |
| --- | --- | --- |
| Command Palette öffnen | `Ctrl+K` | Einige Browser fangen `Ctrl+K` für die Adress- bzw. Suchleiste ab. Passe den Shortcut an, falls die Palette nicht erscheint. |
| Mitglieder ein-/ausblenden | `Ctrl+Shift+M` | Schaltet auf Desktop die rechte Sidebar bzw. mobil das Member-Sheet. |
| Navigation öffnen/schließen | `Ctrl+Shift+D` | Kann in Browsern mit "Alle Tabs als Lesezeichen speichern" kollidieren. |
| Skip to content (Fokus auf Hauptbereich) | `Alt+S` | Setzt den Fokus auf die Haupt-Stage, hilfreich für Tastaturnavigation. |

> Tipp: Leere Eingaben in den Hotkey-Feldern deaktivieren die jeweilige Tastenkombination.
